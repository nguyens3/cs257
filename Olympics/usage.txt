olympics.py usage text

NAME
	olympics - list matching results in database

SYNOPSIS
	python3 olympics.py [-h | --help] 
	python3 olympics.py athletes <noc>
	python3 olympics.py medals
  	python3 olympics.py search_athlete <name>
  
DESCRIPTION
	The following options are available:
  athletes
        Lists all athletes in <string>. If no <string> is given then print usage.
  
  medals
	Lists amount of gold medals of all noc

  -h, --help
	Prints usage statement

  search_athletes
	Lists all of the athletes whose names contain <string> as well as their medals. If no <string> is given then print usage.
